UTSC Daily planner

UTSC Daily planner is a personal project that I concieved of in an attempt to create a product that is useful to university students and to learn more about full stack development. This project's goal was to create a web application that has the following features:

  • A login and signup system using email and password
  • A certain degree of security like encrypt password
  • A task list that has priority listing stored on the server

Things I have learnt:

  • How to prevent sql injection
  • How to encrypt a password using bcrypt
  • How to use prepare statements instead of queries resulting in safer and faster queries
  • Better understanding of how a full stack development and its components work together
